# System Software: Processing Modes Learning Tool

This interactive web application is designed specifically for students studying Computer System Fundamentals. Its goal is to help you clearly distinguish the characteristics, mechanisms, and real-world applications of various computer processing modes.

## 🎯 What You Will Learn
By using this tool, you will understand the differences between:
1. **Batch Processing System (批工件處理系統):** How computers handle massive amounts of data efficiently without human intervention.
2. **Real-time System (實時系統):** Why certain systems guarantee instant responses and how they handle immediate inputs.
3. **Parallel Processing System (並行式處理系統):** How a single computer uses multiple cores to divide and conquer a complex task.
4. **Distributed Processing System (分布式處理系統):** How multiple independent computers over a network collaborate to solve massive problems.
5. **Virtualization (虛擬化):** How one physical machine can act as multiple independent computers using a Hypervisor.

## 🚀 How to Use This App for Learning

### 1. Explore the Learning Modes
*   Use the **Sidebar Navigation** on the left to switch between different processing modes.
*   Read the **Characteristics & Applications** section to understand *why* and *where* each mode is used in the real world.
*   Pay special attention to the **Key Details for Students** box. This section clears up common misconceptions that often appear in exams!

### 2. Interact with the Demos
*   Each mode has an **Interactive Demo**. Click the action buttons (e.g., "Add Task", "Trigger Event") to see a visual representation of how the data flows and is processed.
*   **Observe the buttons:** Notice how some buttons disable themselves. For example, in Batch Processing, you cannot "Start Processing" until you have added data to the queue.
*   **Read the Explanations:** As the animation plays, read the text box below the demo. It explains exactly what is happening step-by-step.

### 3. Test Your Knowledge
*   Once you feel confident, click the **"Module Quiz"** button.
*   The quiz presents real-world scenarios. Choose the correct processing mode for each scenario.
*   After answering, read the detailed explanation provided to reinforce your understanding.

### 4. Bilingual Support
*   Click the **Language Toggle** at the top right (English / 中文) at any time to switch the entire application's language. This is great for familiarizing yourself with technical terms in both languages!
